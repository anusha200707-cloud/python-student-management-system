#Student Management System (CLI)

## About the Project
This is a Command-Line Interface (CLI) based Student Management System developed using Python.  
The application helps manage student records digitally instead of manually.

## Features
- Add student details (Name, ID, Marks)
- View student list
- Search student by ID or Name
- Calculate average marks
- Identify topper

## Technology Used
- Python

## How to Run
1. Download the Python file.
2. Open terminal/command prompt.
3. Run the program:
   python student_management.py
